A visually appealing and responsive Coffee Website built using HTML and CSS, designed to showcase a coffee shop’s offerings and provide a seamless user experience.

** Features**

Homepage – Welcoming visitors with an attractive layout and featured sections.

About Us – Share the story, vision, and values of the coffee shop.

Reservation – Simple form to reserve a table easily.

Review – Display customer reviews and testimonials.

Gallery – Showcase images of coffee, drinks, and ambiance.

Menu – Highlight coffee options, snacks, and special offerings.

** Technologies Used**

HTML5 – Structured and semantic web pages.

CSS3 – Styling, layout design, and responsive elements.
